# LoggerHTTP

**An HTTP logger handler for Elixir applications**

## Installation

The package can be installed by adding `logger_http` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:logger_http, "~> 0.1.0"}
  ]
end
```

The docs can be found at <https://hexdocs.pm/logger_http>.
